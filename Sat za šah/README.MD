# Sat za šah (ATmega328P-AU)

**Vrsta projekta:** Ugradbeni sustav  
**Mikrokontroler:** ATmega328P-AU  
**Jezik:** C / Arduino  
**Alati:** Arduino IDE, alati za dizajn PCB-a (KiCad, EasyEDA)  

## Opis projekta
Ovaj projekt predstavlja elektronički sat za šah razvijen kao cjelovit ugradbeni sustav – od dizajna hardvera do implementacije firmwarea. Uređaj mjeri vrijeme za dva igrača te omogućuje prebacivanje aktivnog timera pomoću fizičkih tipki.

## Moj doprinos
- Izrada elektroničke sheme
- Dizajn PCB layouta
- Sastavljanje i testiranje hardvera
- Implementacija firmwarea za:
  - mjerenje vremena pomoću millis()
  - obradu pritisaka tipki (debounce)
  - logiku prebacivanja igrača
  - komunikaciju s I2C LCD zaslonom

## Stečena znanja
- Rad s mikrokontrolerima i vremenskom logikom
- Dizajn i debugiranje vlastitih PCB pločica
- Rješavanje stvarnih problema u komunikaciji (I2C, napajanje)
- Paralelno rješavanje hardverskih i softverskih grešaka
